# I Close My Eyes (To Open Yours)

*Audio-driven puzzle game inspired by the Orpheus and Eurydice myth. Built in 10 days for a game jam using Godot 4.4.1.*

---

## Description
The protagonist cannot see and navigates the underworld using audio cues.  
Features beat-synchronized audio, tile-based interactions, and NPCs with unique behaviors.

---

## Controls
- W / A / S / D — Move  
- V — Open eyes briefly  

Blind Cane Tile Detection (8 surrounding tiles in clockwise order starting from pressed direction):
- ↑ Up Arrow  
- ← Left Arrow  
- ↓ Down Arrow  
- → Right Arrow  

---

## Features
- Fully audio-driven gameplay  
- Beat-synced audio system (80 BPM)  
- Ghost & Siren NPC behaviors  
- Tile property sound system  
- Five levels with unique music  
- Built in Godot 4.4.1 using GDScript
